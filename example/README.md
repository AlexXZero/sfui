# Project Example

This directory contains an example project demonstrating how to use the library.
The example showcases the library's features and provides a practical reference
for integrating and using it in your own projects.

## Getting Started

To build and run the example, follow these steps:

#### 1. Build the example:

1. Create a build example directory and navigate into it:
   ```sh
   mkdir example/build
   cd example/build
   ```

2. Run CMake to configure the project:
   ```sh
   cmake ..
   ```

3. Build the project:
   ```sh
   make
   ```

#### 2. Run the example:
   ```sh
   ./ExampleProject
   ```
